#  Vizancia

**Learn AI the fun way** — An interactive iOS app that teaches Artificial Intelligence concepts through bite-sized lessons and mini-games.

## Features

- **10 Learning Paths** — From AI basics to future of AI
- **360 Interactive Questions** — 6 question types including multiple choice, fill-in-blank, match pairs, sort order, scenario judgment
- **5 Mini-Games** — Speed Round, AI or Not?, Ethics Court, Prompt Craft, Buzzword Buster
- **Gamification** — XP, levels, streaks, hearts, achievements, daily goals
- **100% Offline** — No internet, no accounts, no data collection
- **Dark Mode** — Full adaptive color system

## Tech Stack

- Swift 5.9+ / SwiftUI
- SwiftData (local persistence)
- iOS 17.0+
- No external dependencies

## Getting Started

1. Clone: `git clone https://github.com/Odin686/Vizancia.git`
2. Open `Vizancia.xcodeproj` in Xcode 15+
3. Select a simulator and press `⌘R`

## Privacy

Vizancia collects zero user data. All progress is stored locally on-device via SwiftData.
